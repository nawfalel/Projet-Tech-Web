Pour lancer l'application sur une distriubtion linux (debian) en étant dans le chemin du projet, exécuter la commande:

    $ ./script_runner.sh

Remarque:

- Vous devez nécessairement avoir sur votre systeme les programmes suivants: **npm** et **java**
